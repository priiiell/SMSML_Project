[Deskripsi Alur Workflow CI]
• Workflow otomatis akan terpicu (trigger) setiap kali ada aktivitas 'push' atau 'pull request' di branch utama (main/master).
• Runner GitHub Actions akan otomatis menyiapkan lingkungan Python (Setup Python Environment).
• Melakukan instalasi seluruh dependensi library yang dibutuhkan secara otomatis melalui file requirements.txt.
• Menjalankan proses pengujian kode atau eksekusi otomatis file pemrograman 'modelling.py' untuk memastikan model dapat dibangun dengan sukses dan lolos uji integrasi (Centang Hijau di GitHub).
